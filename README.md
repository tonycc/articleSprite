# Article Sprite

一个简单的浏览器插件，用于提取网页文章内容并提供沉浸式阅读体验。

## 功能

- 自动提取网页文章的标题和正文内容
- 创建遮罩层进行沉浸式阅读
- 通过右键菜单快速开启阅读模式
- 响应式设计，适配不同屏幕尺寸

## 安装说明

1. 下载或克隆本仓库
2. 在Chrome浏览器中打开 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本插件的目录

## 使用方法

1. 在任意包含文章的网页中，右键点击页面
2. 在弹出的菜单中选择"提取文章内容"
3. 插件会自动提取文章内容并显示在遮罩层中
4. 点击右上角关闭按钮可退出阅读模式

## 注意事项

- 请在icons目录下添加以下图标文件：
  - icon16.png (16x16)
  - icon48.png (48x48)
  - icon128.png (128x128)
